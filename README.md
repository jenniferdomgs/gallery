Aplicação web que exibe uma galeria de fotos, permitindo que os
usuários visualizem, filtrem e pesquisem imagens com base em palavras-chave.

---

## Sobre 
gallery é um projeto desenvolvido como desafio técnico do Processo Seletivo da Aceleradora Ágil, turma 29 (2026/1º Semestre) da PUCRS.

---

## Funcionalidades
- Layout Responsivo
- Barra de Busca Inteligente
- Visualização de Fotos via dados estáticos (podendo consumir uma API pública futuramente)
  
---

## Como rodar (Desenvolvimento)
Instruções genéricas; ajuste conforme a stack do projeto.

1. Clone o repositório
```bash
git clone https://github.com/jenniferdomgs/gallery.git
cd gallery
```

2. Instale dependências
```bash
# usando npm
npm install

# ou usando yarn
yarn
```
3. Executar
```bash
npm run dev
# ou
yarn dev
```

A aplicação ficará disponível em http://localhost:3000
